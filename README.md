# Intellinet Switch

Prototype system that monitors network performance and automatically switches to the optimal connection. Designed to be modular and extendable, supporting multiple metrics such as latency, speed, and packet loss.

## Current Features
- Simulated WiFi vs hotspot latency
- Decision logic for automatic switching
- Console output logs for demonstration

## Next Step(s)
- Improve simulation accuracy with multiple network metrics
- Implement physical switching using Raspberry Pi + LTE module
- Expand to include speed, packet loss, and other network performance differentiators
